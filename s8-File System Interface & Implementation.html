File Concept

  • Computer can store information on serveral different storage media, such as magnetic disks, magnetic tapes, and optical disks.
  • So that the computer system will be convenient to use, the operating system provides a uniform logical view of information storage.
  • The operating system abtracts from the physical properties of its storage devices to define a logical storage unit, the file.

File Types

  • One major consideration in designing a file system, and the entrie operating system, is whether the operating system should recognize and support file types
  • If an operating system recognizes the type of a file, it can then operate on the file in reasonable ways.
  • File types in name (MS-DOS, Windows)
    • Executable (exe,com,bin)
    • Object (obj,o)
    • Source Code (c,p,pas,asm)
    • Text (txt)
    • Archive (zip,tar)
  • File types in attribute (Unix,Linux)

Directory

  • The device directory record information - such as name, location, size, type - for all files on that partition
